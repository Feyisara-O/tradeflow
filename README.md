📊 LedgerPulse — Inventory & Sales Management Platform

A modern, mobile-first Inventory & Sales Management Platform developed as my Final Year Project for Nigerian Small and Medium-sized Enterprises (SMEs).

LedgerPulse helps small businesses manage inventory, monitor stock levels, process sales, and generate insightful reports through a clean and intuitive web interface.

✨ Features

- 🔐 Administrator authentication
- 📦 Product management
- 🏷 Category management
- 📈 Inventory tracking
- 💰 Point of Sale (POS)
- 🧾 Sales history
- 📊 Reports & analytics
- 📉 Low stock monitoring
- 🌙 Light & Dark mode
- 📱 Mobile-first responsive design
- 📡 Progressive Web App (PWA)
- 📥 Installable on desktop and mobile
- ⚡ Offline support using Service Workers

🎯 Project Objectives

The aim of this project is to provide Nigerian retail SMEs with a lightweight inventory and sales management solution that is:

- Easy to use
- Mobile friendly
- Installable as a PWA
- Functional without internet connectivity
- Simple to deploy
